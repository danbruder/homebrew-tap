# danbruder/tap

Homebrew tap for Dan Bruder's CLI tools.

## Installation

```bash
brew tap danbruder/tap
```

## Available Formulae

### trello-cli

A comprehensive Trello CLI tool optimized for LLM integration.

```bash
brew install danbruder/tap/trello-cli
```
